# List

A List in Groovy is an ordered collection of elements. Each element can be of any data type, and they are stored in the order they were added. Lists are useful for storing and manipulating sequences of data.

### Creating Lists

Lists can be created using square brackets `[]` and separating elements with commas:

```groovy
def fruits = ['apple', 'banana', 'orange']
def numbers = [1, 2, 3, 4, 5]
```

### Accessing Elements

You can access elements in a list using their index. Indexing is 0-based, meaning the first element has an index of 0:

```groovy
def colors = ['red', 'green', 'blue']
def firstColor = colors[0]  // 'red'
def secondColor = colors[1] // 'green'
```

### Modifying Lists

Lists are mutable, meaning you can add, remove, or modify elements in them.

- **Adding Elements:**

  ```groovy
  def animals = ['cat', 'dog']
  animals.add('elephant') // ['cat', 'dog', 'elephant']
  ```

- **Removing Elements:**

  ```groovy
  def numbers = [1, 2, 3, 4, 5]
  numbers.remove(3) // Removes the element at index 3
  ```

- **Replacing Elements:**
  ```groovy
  def fruits = ['apple', 'banana', 'orange']
  fruits[1] = 'grape' // Replaces 'banana' with 'grape'
  ```

### Common List Methods

- **`size()`:** Get the number of elements in the list.

  ```groovy
  def colors = ['red', 'green', 'blue']
  def numColors = colors.size() // Returns 3
  ```

- **`contains(element)`:** Check if the list contains a specific element.

  ```groovy
  def animals = ['cat', 'dog', 'elephant']
  def containsDog = animals.contains('dog') // Returns true
  ```

- **`indexOf(element)`:** Find the index of a specific element in the list.

  ```groovy
  def colors = ['red', 'green', 'blue']
  def indexOfGreen = colors.indexOf('green') // Returns 1
  ```

- **`join(separator)`:** Create a string by joining elements with a separator.
  ```groovy
  def words = ['Hello', 'world']
  def sentence = words.join(' ') // Returns 'Hello world'
  ```

### Example

```groovy
def shoppingList = ['apples', 'bread', 'milk']
def numItems = shoppingList.size()
def hasMilk = shoppingList.contains('milk')

println("Shopping List: ${shoppingList}")
println("Number of Items: ${numItems}")
println("Has Milk: ${hasMilk}")
```
